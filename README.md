## 🎯 Objective
The goal of this project is to analyze Blinkit's sales data and identify key trends related to:

- Product categories
- Outlet types and sizes
- Customer preferences
- Sales performance across locations

## 🛠️ Tools & Technologies
- Power BI (Dashboard & Visualization)
- DAX (Data Analysis Expressions)
- Excel / CSV (Data Source)
- Data Cleaning & Transformation

## 📊 Key Features
- Interactive dashboard with real-time filtering

### KPI Tracking:
- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 8523
- Average Rating: 3.9

### Filters:
- Outlet Location Type
- Outlet Size
- Item Type

### Visual Insights:
- Sales by Item Type
- Fat Content Analysis (Low Fat vs Regular)
- Outlet Establishment Trends
- Sales by Outlet Size and Location
- Performance by Outlet Type

## 🔍 Key Insights
- Tier 3 outlets generate the highest sales compared to Tier 1 and Tier 2
- Supermarket Type1 contributes the most to overall revenue
- Fruits, Snacks, and Household items are top-performing categories
- Low Fat products dominate overall sales distribution
- Significant growth observed in outlet establishments over the years
